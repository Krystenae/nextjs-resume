# Next.js Resume Project

## Description
This is a simple resume website built using Next.js. It displays my personal information, skills, and experience in a clean web format.

## Features
- Built with Next.js
- Responsive design
- Simple homepage layout
- Easy to update content

## How to Run
1. Install dependencies:
   npm install

2. Start development server:
   npm run dev

3. Open in browser:
   http://localhost:3000
